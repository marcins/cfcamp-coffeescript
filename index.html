What is CoffeeScript?

“A better way of writing JavaScript”

Alt.JS

Like TypeScript

Not like ClojureScript, Roy, etc

It's a transpiler!

So what is it?

  • It's like a mix of Ruby, Python and Javascript
  • No curly braces
  • (Mostly) optional parentheses
  • Syntactically significant whitespace
It's like a mix of Ruby, Python and Javascript. No curly braces. (Mostly) optional parentheses. Syntactically significant whitespace.

Why not just use JavaScript?

  • It's just more fun to write CoffeeScript!
  • It can be easier to read
  • No technical reason

So what are the drawbacks?

  • More difficult to debug (Source Maps help)
  • Requires a compile step (Tools make this less painful)
  • Potential incompatibilities with ES6 Harmony functionality
  • It's not a substitute for knowing JavaScript!

Ranges: Splicing

cities[0..1]

is equivalent to

cities[..1]

You can use negative indexes:

cities[-1..]

As you can see all these options translate to simple Javascript array splices

As a nice bonus you can also replace the result of a slice:

cities[-1..] = "New York"

Using CoffeeScript

Installing CoffeeScript

npm install -g coffee-script

Running CoffeeScript

coffee -c file.coffee # compile
coffee -l file.coffee # lint
coffee -wc file.coffee # watch & compile
coffee -cm file.coffee # sourcemap
coffee -wmlc *.coffee # ALL THE THINGS!
